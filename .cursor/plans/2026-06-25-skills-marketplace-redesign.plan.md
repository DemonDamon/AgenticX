# 技能市场折叠面板重构

Planned-with: （待补充）

## 目标

将 Desktop 设置页「技能配置」中「官方推荐 / CLAWHUB 市场 / SKILLHUB 市场」三块整合为可折叠的「技能市场」面板，视觉与上方「全局技能 / 扫描路径」一致，并减少垂直占用。

## 范围

- 文件：`desktop/src/components/SettingsPanel.tsx`
- 不改技能列表、扫描路径、扩展包等上方/下方既有区块

## 需求

- FR-1: 新增可折叠「技能市场」Panel，默认与全局技能同级样式
- FR-2: 内含三个子区块：官方推荐、CLAWHUB 市场、SKILLHUB 市场
- FR-3: 子区块使用卡片式背景区分（`bg-surface-panel` + border）
- FR-4: 官方推荐宽屏最多 3 列（`md:grid-cols-3`）
- FR-5: 移除区块级说明文案，卡片内不展示长描述

## 验收

- AC-1: 「技能市场」可折叠/展开，样式与「全局技能」一致
- AC-2: 三块内容均在折叠面板内，无额外顶部分割横线
- AC-3: 宽屏下官方推荐最多 3 列
- AC-4: 官方推荐卡片无长段说明文字
